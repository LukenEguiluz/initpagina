# Cómo nombrar las fotos de empleados

Coloca las fotos en **`public/empleados-fotos/`** (ahí la web las sirve). Usa los nombres indicados abajo. Usa **exactamente** estos nombres de archivo para que la aplicación reconozca a cada persona.

---

## Reglas del nombre

- Formato: **apellido(s)-nombre(s).extension**
- Minúsculas, **sin tildes** en el nombre del archivo (ej. `inaki`, no `iñaki`).
- Guiones `-` entre apellidos y entre apellido y nombre.
- Extensiones válidas: **`.jpg`**, **`.jpeg`** o **`.png`**.

---

## Cómo debe quedar la carpeta

```
public/empleados-fotos/
├── jimenez-guevara-enrique.jpg
├── guerrero-negrete-inaki.jpg
├── corona-del-rio-javier.jpg
├── eguiluz-del-angel-luken.jpg
├── martinez-carolina.jpg
└── rodriguez-xoan-pablo.jpg
```

Solo tienes que guardar cada foto con el nombre de la tabla y ponerla en esta carpeta.
